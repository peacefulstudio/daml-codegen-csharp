# Daml.Codegen.Testing.Conformance

Compiled C# generated by `daml-codegen-csharp` from the conformance corpus,
plus the corpus DAR embedded as a resource. Consumers (the canton-ledger-api
integration tests) round-trip these types against a real Canton participant to
prove codegen output works end-to-end. Not for production use.

`ConformanceCorpus.OpenDar()` returns the embedded DAR to upload to a participant.
